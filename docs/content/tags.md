# Tags

Each content item can have multiple tags:

- **movie**. Description: Feature film, Color: red
- **series** - Description: TV series, Color: green
- **anime** - Description: Anime content, Color: sea blue
- **feature** - Description: Full-length (>40 minutes), Color: Brown
- **short** - Description: Short film (<40 minutes), Color: grey
- **cartoon** - Description: Cartoon, Color: pink

This is a list default tags, but they might me changed
